# API Spec

## Register User API

Endpoint : POST /api/user/register

Request Body :

```json
{
  "username" : "example",
  "email" : "example@mail.com",
  "password" : "example123"
}
```

Response Body Success

```json
{
  "message" : "Registration Successful",
  "data" : ""
}
```
